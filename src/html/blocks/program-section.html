Программа обучения

*План обучения составляется индивидуально

Примерная программа обучения

1. Основы HTML и CSS

Изучение структуры веб-страницы (HTML) и стилизации (CSS). Основные теги, классы, ID. Практика создания простых веб-страниц.

2. Продвинутые CSS техники

Flexbox, Grid, адаптивная верстка. Анимации и трансформации для улучшения внешнего вида страниц.

3. Основы JavaScript

Знакомство с JavaScript: переменные, функции, массивы, условия. Практика создания интерактивных элементов на сайте.

4. Работа с DOM

Управление и модификация HTML и CSS через JavaScript. Работа с событиями и динамическое изменение контента.

5. Основы Git и GitHub

Знакомство с системами контроля версий: создание репозиториев, коммиты, ветвление и слияние. Публикация кода на GitHub.

6. Введение в CSS препроцессоры

Изучение Sass или Less для оптимизации написания стилей. Использование переменных, вложенных правил и миксинов.

7. Основы JavaScript фреймворков

Изучение базового функционала популярных фреймворков: React, Next.js. Компоненты, состояние и управление данными.

8. UI библиотеки

Знакомство с популярными UI библиотеками, такими как MUI, Bootstrap, Ant.d и другие для ускорения верстки. Использование готовых компонентов и сеток для адаптивного дизайна.

9. API и работа с сервером

Получение и отправка данных через API с использованием fetch или axios. Знакомство с REST API и JSON.

10. Проектная работа

Финальные проекты: создание веб-приложения или вебсайта с использованием HTML, CSS, JavaScript и фреймворка.

Как проходит урок

На уроке мы пишем код в режиме реального времени — решаем задачи, создаем странички и приложения. Благодаря такому подходу лайвкодинг на собеседованиях больше не пугает. Используется программа удаленного доступа, где я могу корректировать код ученика.

Подведение итогов

Время от времени происходит итоговый урок для проверки знаний и понимания того, насколько хорошо ученик усвоил программу. На таком уроке присутствуют вопросы и задачи по всем ранее изученым темам.

Помощь в трудоустройстве

Помогаю ученикам сделать качественное резюме и написать универсальный cover letter. Создаем и оформляем профиль в линкедин. Учимся отвечать на каверзные вопросы HR'ов :).
